# 🎬 AILifeMovie

AILifeMovie is a lightweight AI-powered web app that transforms personal memories into cinematic 60-second life story narrations.

Users can paste their life experiences—such as childhood moments, travel stories, achievements, and personal milestones—and the app uses AI to turn them into emotional, movie-trailer-style storytelling.

The goal is to reimagine personal memories as powerful cinematic narratives.

---

## ✨ Features

- 🧠 Convert raw life memories into cinematic AI-generated stories  
- 🎬 Emotional storytelling in movie trailer style  
- ⚡ Fast and simple single-page interface  
- 📋 One-click copy of generated story  
- 🎨 Clean, minimal UI focused on user experience  

---

## 🛠️ Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- OpenAI API

---

## 💡 How It Works

1. User pastes personal memories into the text box  
2. Frontend sends the input to a Next.js API route  
3. OpenAI generates a cinematic narration based on the input  
4. The app displays the final emotional story for the user  

---

## 🚀 Purpose

This project is built as a simple AI demo to explore how generative AI can transform personal text into emotional, narrative-driven content. It showcases the potential of AI in creative storytelling and personal expression.

---

## 📌 Note

This is a minimal MVP project and does not include authentication, database, or persistence. It is designed for demonstration and experimentation purposes only.
